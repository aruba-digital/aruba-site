# aruba-site — Site Corporativo Aruba Digital

Site corporativo de Aruba Digital. En desarrollo.

## Info

- URL producción: arubadigital.cl (pendiente de publicar)
- URL dev: http://localhost:3001
- Empresa: Aruba Digital SPA (en proceso de constitución)

## Stack

- Next.js 16 + TypeScript
- Chakra UI v3 + Emotion
- Framer Motion
- Tailwind CSS v4

## Comandos

```bash
npm run dev      # Servidor de desarrollo (localhost:3001)
npm run build    # Build de producción
npm start        # Servidor de producción
npm run lint     # ESLint
```

## Estructura

```
src/
├── app/         # Rutas Next.js (App Router)
├── components/  # Componentes UI
└── theme/       # Tema de Chakra UI (aruba-theme)
```
